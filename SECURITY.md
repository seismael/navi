# Security Policy

## Supported Versions

Currently, only the latest version of Navi is supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| Main    | :white_check_mark: |

## Reporting a Vulnerability

We take the security of Navi seriously. If you believe you have found a security
vulnerability, please report it to us by following these steps:

1.  **Do not** open a public issue.
2.  Send an email to [INSERT EMAIL ADDRESS] with a description of the
    vulnerability.
3.  Include as much information as possible, including steps to reproduce the
    issue and any potential impact.

We will acknowledge receipt of your report and provide a timeline for addressing
the issue. We ask that you follow responsible disclosure practices and give us
adequate time to fix the vulnerability before making it public.

## Security Updates

Security updates will be released as part of regular releases or as emergency
patches if necessary. We recommend staying up to date with the latest version
of Navi to ensure you have the latest security fixes.
